# House Price Prediction

## Project Overview
This project uses a Linear Regression model to predict house prices based on various housing features. The dataset was preprocessed by handling missing values and preparing features for training.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Model Evaluation
- RMSE: 52975.72
- R² Score: 0.6341

## Features
- Data preprocessing
- Model training using Linear Regression
- Performance evaluation using RMSE and R² Score
- Data visualization

## Dataset
Kaggle House Prices Dataset
